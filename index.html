Canna - Japanese input system

Overview

Canna is a Japanese input system for UNIX/Linux. It consists of a kana-to-kanji conversion engine (cannaserver), a user interface engine (libcanna) and a front end processor for console (canuum). You can also use Canna in X11 (via "kinput2" XIM server), Emacs (with an appropreate patch) or other applications.

Download

The latest release is Canna 3.7p5 (released on 15 Aug 2025). Older releases are available at the GitHub release page. Development version on GitHub can be retrieved by:
git clone https://github.com/canna-input/canna
and run mkrelease.sh at the top of the source tree.

Development

GitHub project page is here. Being designed in 1990s, this project is mostly for maintenance and distribution of existing releases. If you have any patches or bug reports, submit it into GitHub Issues or mail here.
