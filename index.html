Canna - Japanese input system

Overview

Canna is a Japanese input system for UNIX/Linux. It consists of a kana-to-kanji conversion engine (cannaserver), a user interface engine (libcanna) and a front end processor for console (canuum). You can also use Canna in X11 (via "kinput2" XIM server), Emacs (with an appropreate patch) or other applications.
